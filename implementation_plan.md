# THE K - Web Site Implementation Plan

## 概要
「THE K」のブランドイメージに合わせた、高級感のあるWebサイトを構築しました。
HTML, CSS, JavaScript (GSAP) を使用し、洗練されたアニメーションとデザインを実現しています。

## 技術スタック
- **HTML5**: サイトの骨組み。SEOを意識したセマンティックな構造。
- **CSS3**: デザインとレイアウト。変数やClamp関数を使用したレスポンシブ対応。
- **JavaScript (Vanilla)**: 動的な振る舞い。
- **GSAP (GreenSock Animation Platform)**: 高品質なアニメーションライブラリ。
- **Vite**: 高速な開発環境とビルドツール。

## 実装内容
1.  **デザインシステム**:
    - カラー: ゴールド (#C5A059), オフホワイト (#F9F8F4), ブラック (#1A1A1A)
    - フォント: 英語は「Cormorant Garamond」、日本語は「Shippori Mincho」を使用し、和モダンな雰囲気を演出。
2.  **アニメーション**:
    - **ローディング**: サイトを開いた瞬間にロゴが浮かび上がり、幕が開くような演出。
    - **スクロール演出**: スクロールに合わせて文字や画像がふわっと現れる演出。
    - **パララックス**: ヒーロー画像の背景がスクロール速度とずれて動くことで奥行きを表現。
3.  **セクション構成**:
    - Hero: インパクトのあるメインビジュアル。
    - Concept: 縦書きテキストを用いた日本的な哲学の表現。
    - Values: 3つのコア価値をカード形式で紹介。
    - Plan: タイムライン形式でのモデルプラン紹介。
    - Flow: 予約までの流れをステップ表示。
    - Contact: 予約への導線。

## 使い方 (中学生向け解説)

### 1. 開発サーバーを起動する
サイトを自分のパソコンで確認しながら編集するには、以下のコマンドを使います。
```bash
npm run dev
```
これを実行すると、黒い画面にURL（例: `http://localhost:5173`）が表示されます。そのURLをブラウザで開くと、作ったサイトが見れます。
ファイルを保存すると、自動で画面が更新されるので便利です！

### 2. 公開用のファイルを作る
インターネットに公開するための「完成品」を作るには、以下のコマンドを使います。
```bash
npm run build
```
これを実行すると、`dist` というフォルダの中に、圧縮されたファイルが出来上がります。これをサーバーにアップロードすれば、世界中の人が見られるようになります。

### 3. コードを編集する
- **文字を変えたい**: `index.html` の中の文字を書き換えてください。
- **色や大きさを変えたい**: `style.css` を編集します。`:root` の部分にある色コードを変えると、サイト全体の色が一括で変わります。
- **動きを変えたい**: `main.js` を編集します。`duration`（時間）や `delay`（遅延）の数字を変えると、アニメーションのタイミングが変わります。
